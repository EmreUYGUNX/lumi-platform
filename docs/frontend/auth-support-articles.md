# Customer Support Articles (Auth)

## Reset Password

- Go to **Forgot password** on the login page.
- Enter your email; check inbox (and spam) for the reset link.
- Follow the link, set a 12+ character password with upper/lowercase, number, and symbol.

## Verify Email

- After registering, open the verification email and click the link.
- If missing: use **Resend verification** in the app or check spam.
- Without verification, some features remain limited.

## Manage Sessions

- Navigate to **Account → Sessions**.
- Review active devices; revoke unknown sessions.
- Use **Revoke all** to force logout everywhere; then log back in.

## Enable 2FA (Roadmap)

- Two-step verification will arrive in a future release.
- Until then, keep a strong password and watch security alerts.

## Request Data Export

- Go to **Account → Privacy**.
- Click **Request data export**; you’ll receive a download link via email when ready.
